# Portfolio Intelligence

Execution Stability Index — Portfolio Comparison

Compare ESI scores across workstreams and programs to surface which portfolio areas are trending toward systemic risk and require priority intervention. Part of the Program Intelligence discipline: https://krayu.be/program-intelligence

---

## Overview

**Execution Stability Index (ESI)** is a program intelligence signal measuring the stability and regularity of engineering execution patterns across time. ESI is derived from five execution dimensions:

- Schedule Stability
- Cost Stability
- Risk Acceleration Gradient
- Delivery Predictability
- Flow Compression

## Intervention Thresholds

- **P1 — Urgent**: ESI below 55 — Priority Intervention Required
- **P2 — Watch**: ESI 55–69 — Monitor Closely, Early Intervention Recommended
- **P3 — Monitor**: ESI 70–84 — Maintain Oversight
- **P4 — Stable**: ESI 85+ — Structurally Stable, High Resilience

## Scenario Simulation — Model Logic

- Budget injection → affects Cost Stability, Delivery Predictability
- Timeline extension → affects Schedule Stability, Flow Compression
- Resource increase → affects Delivery Predictability, Flow Compression
